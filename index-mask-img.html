Kaneki en modo Mask-Image

Máscara en Alpha

La composición alfa o canal alfa es la que define la opacidad de un píxel en una imagen. El canal alfa actúa como una máscara de transparencia que permite, de forma virtual, componer (mezclar capas) imágenes o fondos opacos con imágenes con un cierto grado de transparencia.

    .alpha-target {
  mask-image: url(https://cdn.pixabay.com/photo/2017/07/06/18/45/fidget-spinner-2478943_960_720.png);
  mask-mode: alpha;
  mask-repeat: no-repeat;
  mask-size: 80%;
}

Máscara Luminiscencia

El canal luminance es la que define el brillo de un píxel en una imagen. Actúa como una máscara de luz que permite jugar con la luminosidad de las imágenes.

    .luminance-target {
  mask-image: url(https://cdn.pixabay.com/photo/2017/07/06/18/45/fidget-spinner-2478943_960_720.png);
  mask-mode: luminance;
  mask-repeat: no-repeat;
  mask-size: 80%;
}

Se repite la Mask varias veces

El atributo mask-repeat lo que hace es repetir la máscara sobre la imagen en un ángulo.

    .target-repeat {
  margin: 20px auto;
  display: block;
  max-width: 100%;
  height: 450px;
  mask-image: url(https://cdn.pixabay.com/photo/2017/07/06/18/45/fidget-spinner-2478943_960_720.png);
  mask-repeat: space;
  mask-size: 100px 100px;
}

Para el posicionamiento utilizamos

    .mask-container {
  border: 5px solid #444;
  width: 80%;
  margin: 0 auto;
  padding: 0;
}

Máscara en lateral izquierdo-superior

Crea la máscara en una posición indicada

    .mask-pos-top .mask-pos-target {
  mask-position: 0 0;
}

Máscara en lado derecho-inferior

Crea la máscara en una posición indicada

     .mask-pos-bottom .mask-pos-target {
  mask-position: 100% 100%;
}

Para el tamaño usamos

  .mask-size-target {
  mask-image: url(https://cdn.pixabay.com/photo/2017/07/06/18/45/fidget-spinner-2478943_960_720.png);
  mask-repeat: no-repeat;
  mask-position: center center;
}

Propieda de máscara a mitad de la imagen

Posiciona la máscara en el centro de la imagen

    .mask-size-half .mask-size-target {
  mask-size: 50%;
}

Propiedad de máscara a la mitad del contenedor

    .mask-size-contain .mask-size-target {
  mask-size: contain;
}

Posiciona la máscara en el centro del contenedor
